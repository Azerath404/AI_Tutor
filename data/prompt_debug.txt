Khai báo con trỏ
❖Sử dụng từ khóa typedef
❖Ví dụ
❖Lưu ý khi khai báo kiểu dữ liệu mới

typedef<kiểu dữ liệu> *<tên kiểu con trỏ>;
<tên kiểu con trỏ> <tên biến con trỏ>;
typedefint *pint;
int *p1;
pint p2, p3;

Con trỏ NULL
❖Khái niệm
▪Con trỏ NULLlà con trỏ không trỏ và đâu cả.
▪Khác với con trỏ chưa được khởi tạo.
Dữ liệu kiểu con trỏ
NULL
int n;
int *p1 = &n;
int *p2; // unreferenced local variable
int *p3 = NULL;

Khai báo con trỏ
❖Sử dụng từ khóa typedef
❖Ví dụ
❖Lưu ý khi khai báo kiểu dữ liệu mới

int *p1;
pint p2, p3;

Con trỏ NULL
❖Khái niệm
▪Con trỏ NULLlà con trỏ không trỏ và đâu cả.
▪Khác với con trỏ chưa được khởi tạo.
Dữ liệu kiểu con trỏ
NULL
int n;
int *p1 = &n;
int *p2; // unreferenced local variable
int *p3 = NULL;

Khai báo con trỏ
❖Sử dụng từ khóa typedef
❖Ví dụ
❖Lưu ý khi khai báo kiểu dữ liệu mới
▪Giảm bối rối khi mới tiếp xúc với con trỏ.
▪Nhưng dễ nhầm lẫn với biến thường.
Dữ liệu kiểu con trỏ
typedef<kiểu dữ liệu> *<tên kiểu con trỏ>;
<tên kiểu con trỏ> <tên biến con trỏ>;
typedefint *pint;
int *p1;
pint p2, p3;

Con trỏ NULL
❖Khái niệm
▪Con trỏ NULLlà con trỏ không trỏ và đâu cả.
▪Khác với con trỏ chưa được khởi tạo.
Dữ liệu kiểu con trỏ
NULL
int n;
int *p1 = &n;
int *p2; // unreferenced local variable
int *p3 = NULL;
